# B站缓存视频下载工具

一款用于从 Android 设备下载 B站缓存视频的桌面应用程序。

## 功能特点

- 支持有线/无线 ADB 连接 Android 设备
- 自动扫描设备上的 B站缓存视频
- 批量下载视频到本地
- 支持多P视频下载和播放
- 深色/浅色主题切换
- 封面预览和缩放
- 下载历史记录
- 统计信息展示

## 系统要求

- Windows 10 或更高版本
- 已安装 ADB (Android Debug Bridge)
- Android 设备需开启 USB 调试模式

## 安装方法

### 方式一：安装包安装（推荐）

1. 下载最新版本的安装包 `bili-cache-get-2.1.2-setup.exe`
2. 双击运行安装程序
3. 按照安装向导完成安装
4. 安装完成后会自动创建桌面快捷方式

### 方式二：便携版

1. 下载 `bili-cache-get` 文件夹
2. 双击运行 `bili-cache-get.exe` 即可

## 基本使用步骤

### 1. 连接设备

**有线连接：**
- 使用 USB 数据线连接 Android 设备到电脑
- 确保设备已开启 USB 调试模式
- 在设备上授权此电脑进行调试

**无线连接：**
- 确保设备和电脑在同一局域网
- 在设备上开启无线调试
- 点击"无线连接"按钮，输入设备 IP 地址和端口

### 2. 选择设备

- 在左侧设备列表中选择已连接的设备
- 系统会自动检测设备是否安装 B站客户端
- 选择设备后会自动加载视频列表

### 3. 浏览视频

- 视频列表会显示所有缓存的视频
- 可以通过搜索框筛选视频
- 双击标题列可快速下载单个视频
- 点击封面可预览大图

### 4. 下载视频

- 勾选需要下载的视频
- 点击"下载选中"按钮
- 选择保存位置
- 等待下载完成

### 5. 播放视频

- 下载完成的视频会显示"已完成"状态
- 点击状态按钮可直接打开视频
- 多P视频可选择播放指定分P

## 常见问题

### 设备连接失败

1. 检查 USB 数据线是否正常
2. 确认 USB 调试已开启
3. 尝试重新授权 USB 调试
4. 重启 ADB 服务：`adb kill-server && adb start-server`

### 未检测到 B站客户端

1. 确认设备已安装 B站客户端
2. 确认 B站客户端版本为最新版
3. 尝试在 B站中缓存一些视频

### 下载失败

1. 检查磁盘空间是否充足
2. 确认保存路径有写入权限
3. 尝试重新下载
4. 查看错误提示获取详细解决方案
