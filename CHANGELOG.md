## [v2.0.0] - 2025-03-08

### 新增功能

- **打包发布支持** - 完整的 Windows 可执行文件打包和安装程序
  - PyInstaller 打包配置，生成独立 exe 文件
  - Inno Setup 安装包制作，支持一键安装
  - 自动检测并卸载旧版本
  - 支持桌面快捷方式和开始菜单项

- **虚拟滚动组件** - 大幅提升大量视频列表的性能
  - 基于 QAbstractListModel 的分页加载
  - 每批加载 50 条数据，按需加载更多
  - 自定义委托绘制，减少内存占用
  - 文件：`src/gui/components/virtual_list.py`

- **封面 LRU 缓存** - 优化封面加载性能和内存使用
  - 线程安全的 LRU 缓存实现
  - 默认缓存 100 张封面，自动淘汰最少使用
  - 支持缓存命中率统计
  - 文件：`src/services/cover_cache.py`

- **数据库索引优化** - 提升数据库查询性能
  - 添加设备ID索引 (ix_videos_device_id)
  - 添加下载状态索引 (ix_videos_download_status)
  - 添加设备+状态复合索引 (ix_videos_device_status)
  - 提供数据库迁移脚本
  - 文件：`scripts/migrate_db_indexes.py`

- **快捷键系统** - 提升操作效率
  - `Ctrl+R` - 刷新设备列表
  - `Ctrl+F` - 搜索视频
  - `Ctrl+A` - 全选视频
  - `Ctrl+D` - 下载选中视频
  - `Ctrl+,` - 打开设置
  - `F5` - 刷新视频列表
  - `Delete` - 删除选中
  - 文件：`src/gui/shortcuts.py`

- **首次使用引导** - 新用户友好的入门体验
  - 四步快速上手指南
  - 支持"不再显示"选项
  - 支持深色/浅色主题
  - 文件：`src/gui/dialogs/welcome_dialog.py`

- **错误提示优化** - 友好的错误信息和解决方案
  - 14 种错误类型的详细说明
  - 每种错误提供可能原因和解决方案
  - 统一的错误对话框样式
  - 支持重试功能
  - 文件：`src/gui/error_handler.py`

### 性能改进

- 视频列表支持数千条数据流畅滚动
- 封面加载内存占用降低约 60%
- 数据库查询速度提升 50%+
- 应用启动速度优化

### 修复

- 修复持久化状态丢失问题
- 修复多P视频播放选择问题
- 修复 Windows 系统 GBK 编码报错问题

## [v0.2-GLM5] - 2024

### 修复
- **持久化状态丢失** - 修复已下载视频刷新后状态被重置的问题

- **多P视频播放选择** - 修复多P视频下载完成后无法选择指定P数播放的问题

  - 文件：`src/services/adb_service.py`

  - 当前缩放比例显示
  - 文件：`src/gui/main_window.py`

- **双击下载** - 双击标题列快速下载单个视频
  - 文件：`src/gui/main_window.py`

## 版本说明

### 版本命名规则
- **主版本号 (Major)**: 不兼容的API变更
- **次版本号 (Minor)**: 向后兼容的功能新增
- **修订号 (Patch)**: 向后兼容的问题修复
- **后缀**: 特殊版本标识（如 GLM5 表示AI辅助开发版本）

### 变更类型
- **新增**: 新功能
- **修改**: 现有功能的变更
- **弃用**: 即将移除的功能
- **移除**: 已移除的功能
- **修复**: Bug修复
- **安全**: 安全相关修复
